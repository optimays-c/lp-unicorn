  • Image Classification

    Image classification is the process of taking an input image and outputting it's class with some probability e.g. "there’s a 90% probability that this input is a cat".

  • Object Detection

    Object Detection is the process of taking an input image and localizing and identifying multiple objects in it with the probability that of each object belonging to a particular class.

  • Object Tracking

    Object Tracking is the process of taking a video input and tracking an object which could be a person, ball or a car across a series of it's frames.

  • Semantic Segmentation

    Semantic Segmentation is the process of taking an input image and associating it's each pixel with a class label (such as flower, person, road, sky, ocean, or car).

  • Pose Detection

    Pose Detection is the process to estimate the pose of a person in an image or video by estimating where key body joints are.

  • Audio based classification

    It is a process that utilizes the discriminative capability of the featuers representing acoustic instability, speaker interactivity, speech quality and acoustic space characterization etc. for doing the analysis.

  • Text Classification

    Text Classification is the process of assigning tags or categories to text according to its content. It has broad applications in sentiment analysis, topic labeling, spam detection, intent detection etc.

  • Anomaly Detection

    The process to identify rare items, events or observations which raise suspicions by differing significantly from the majority of the data.

  • Text Summarization

    Text summarization is the process to shorten long pieces of text to outline only the main points of the text.

  • Question Answer Bot

    An intelligent bot that automatically and instantly resolves your customers' most common questions.

  • Time Series Forecasting

    Time series forecasting is the process to predict future values based on the meaningful statistics and other characteristics of the previously observed values.

  • Recommendation System and many more
